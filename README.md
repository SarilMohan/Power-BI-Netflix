 ## Netflix Content Analysis Dashboard — Power BI Project
This project explores Netflix's catalog using data analytics to uncover insights related to content volume, genre distribution, release trends, and country-level contributions. The analysis is visualized through a modern Power BI dashboard designed to present insights clearly and interactively.

## Project Overview
The goal of this project was to transform raw Netflix dataset into meaningful visual insights. Using Power BI, I cleaned the data, modeled relationships, built DAX measures, and created an intuitive dashboard that highlights patterns in Netflix’s content library.

## Tools & Technologies
- **Power BI**
- **Power Query Editor**
- **DAX (Measures & Calculations)**
- **Excel (for initial dataset handling)**
  
## Dashboard Features
- **Total Content Count:** 6131 Titles  
- **Content Type Distribution:**
  - Movies: 6131
  - TV Shows: 2676
- **Top Contributing Countries**
- **Genre Frequency Analysis**
- **Trend of Content Added Over the Years**
- **Comparison Between Movie & TV Show Releases**
- **Filter Panel with Year Selector**

## Key Insights
- Netflix's library is **dominated by movies**, though TV content has increased significantly in recent years.
- The dataset shows a rapid spike in content additions around **2015–2020**, aligning with global expansion.
- **United States, India, Japan, and the UK** are among the top contributors.
- Popular genres include **Documentary, Drama, Comedy, Action, and Kids' TV.**
